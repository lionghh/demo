# 文本
> 文本的字体大小及颜色等，html字体大小为.15rem，颜色为#666，行高为1.5，显示宽度为375px，字体大小为15px

## 字体大小
> ipu-txt-m同body字段大小设置为.15rem

- ipu-txt-s, 
- ipu-txt-m, 
- ipu-txt-l, 
- ipu-txt-xl, 
- ipu-txt-xxl,
    
## 字体颜色

- ipu-txt-hightlight
- ipu-txt-warning
- ipu-txt-success

- ipu-txt-black     // #000

- ipu-txt-default // #666 同body默认设置

- ipu-txt-light     // #ccc
- ipu-txt-white   // #000
