模板页

  • 推荐使用栅格中介绍的样式来布局页面，其优势相比较传统css布局，flex布局是动态化的 通常一个典型的页面分为标题块，内容块 下面将主要介绍以下几个class类及作用 
    <body>
    <div class="ipu-row-flex ipu-row-flex-ver">
        <div class="ipu-col ipu-col-auto">
            顶部
        </div>
        <div class="ipu-col">
            内容区
        </div>
        <div class="ipu-col ipu-col-auto">
            底部
        </div>
    </div>
</body>

    ipu-row-flex

    将元素声明为flex展示，元素的宽度是100%，默认flex元素子元素是纵向排列的

    ipu-row-flex-ver

    ipu-row-flex的修饰，配合ipu-row-flex使用，此class声明元素子元素垂直排列，元素的高度是100%

    ipu-col

    ipu-col作为ipu-row-flex的直接子元素，与其它ipu-col子元素平均分配父元素剩余的空间

    ipu-col-auto

    是ipu-col的修饰，配合ipu-col使用，元素占据空间由其内容本身决定，不参与分配父元素剩余空间
